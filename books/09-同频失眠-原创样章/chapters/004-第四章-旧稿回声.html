第四章 废稿回带

第四章把误会源头改成旧录音和废稿回带，不再走同住照顾线。情绪冲突来自他保存多年的声音存档，以及她对那份心意的后知后觉。

01

第四天晚上，许知遥独自在机房找底稿，意外翻到一只旧的 storage drive，像从最深处的 archive shelf 里突然摸出来。标签边缘已经卷起，手写的 file tag 旁边，原本的 catalogue code 和 metadata 都快看不清了。她把它从最底层的 draft folder 里顺着 retrieval path 抽出来时，心跳忽然慢了一拍。

02

盘里不只有这一周的素材，还有她大学毕业前录过却没播出的 monologue track。每条 outtake 后面都跟着一行细小的 side note，像落在 margin note 里。连她哪一句因为紧张而出现 false start、哪一秒的 falter beat 最明显，周聿白都完整 retain 着。

03

许知遥第一反应不是感动，而是难堪。她以为这档节目里那些恰到好处的默契，只是他提前翻过太多旧素材，带着一点过分冷静的 archive bias，才能把每一次对视和停顿都做得像命中注定。那些被单独分出来的 back catalog，忽然让她觉得自己这几天的心动，也许只是一次过分自作多情的 replay，甚至是一场糟糕的 overread。她甚至想立刻替这份存档做一次 delete pass，彻底把它 purge。

04

周聿白回来时，她正坐在调音台 console 前发怔。男人看见那只硬盘，沉默了一会儿才说，那不是项目资料，而是他的 private folder，一套只对他自己 restricted 的 source file。这些年他换过很多节目，唯独没有把她那段声音从里面挪出去，也没有动过那份漫长的 retention。

05

他按下回放，耳机里立刻浮起一层轻微的 tape hiss，也带着一点旧设备特有的 analog warmth。二十二岁的许知遥在录音里停顿了很久，才对着空气说了一句没来得及寄出去的晚安，像一段迟到了很多年的 unsent script。周聿白低声解释，那句被他写进过很多夜间节目的 liner note，也是他这些年一直没有改掉的 sign-off 习惯，那点喜欢甚至一路 carry over 到今天，仍旧很 persistent。
